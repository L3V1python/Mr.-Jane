# MR. JANE

## Open Source Intelligence Username Investigation Tool

Mr. Jane is a Python-based OSINT username investigation tool designed to search publicly available username profiles across multiple online platforms.

Inspired by the observation and deduction skills of Patrick Jane from *The Mentalist*, Mr. Jane focuses on connecting public information and discovering possible digital footprints across the web.

---

# Features

- Multi-platform username searching
- Automated public profile checking
- Terminal-based interface
- Custom command system
- Expandable platform database
- Lightweight Python architecture
- Fast username discovery
- Easy platform expansion
- Randomized request delays
- Cross-platform support

# Installation

## Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/Mr-Jane.git
```

## Enter the Directory

```bash
cd Mr-Jane
```

## Install Requirements

```bash
pip install -r requirements.txt
```

## Run Mr. Jane

```bash
python3 mrjane.py
```

---

# Usage

Launch the tool:

```bash
python3 mrjane.py
```

Search a username:

```text
Mr.Jane > search username123
```

Example output:

```text
Searching for: username123

[+] Found GitHub:
https://github.com/username123

[+] Found Reddit:
https://www.reddit.com/user/username123

[-] Not found Instagram

[-] Not found TikTok
```

---

# Commands

| Command | Description |
| --- | --- |
| `search <username>` | Search a username across platforms |
| `help` | Display available commands |
| `exit` | Close Mr. Jane |
